# 📋 משימות משפחה

אפליקציית ניהול משימות משפחתית — כל אחד רואה את המשימות שלו, ואמא מאשרת ביצוע.

## איך זה עובד

- **אמא** — רואה את כולם, מוסיפה משימות, מאשרת ביצוע
- **שאר המשפחה** — רואים רק את שלהם, מסמנים ✓ כשסיימו
- משימה נעלמת רק אחרי אישור אמא
- כל שינוי מתעדכן אוטומטית בין כרטיסיות פתוחות

## הפעלה מקומית

פתחי את `index.html` ישירות בדפדפן — לא צריך שרת.

## GitHub Pages

1. צרי ריפוזיטורי חדש ב-GitHub
2. העלי את שלושת הקבצים (`index.html`, `style.css`, `app.js`)
3. לכי ל־Settings → Pages → Source: main branch
4. אחרי כמה שניות האפליקציה זמינה בכתובת `https://USERNAME.github.io/REPO_NAME`

## שינוי שמות וצבעים

ערכי את `DEFAULT_USERS` בתחילת `app.js`:

```js
const DEFAULT_USERS = [
  { id: 'shimi', name: 'אמא', emoji: '👩', color: '#9FE1CB', border: '#1D9E75', admin: true },
  // ...
];
```

## שמירת נתונים

הנתונים נשמרים ב-`localStorage` של הדפדפן.  
לשיתוף אמיתי בין מכשירים שונים — צריך להחליף ל-backend (Firebase, Supabase וכו׳).

## קבצים

| קובץ | תיאור |
|------|--------|
| `index.html` | מבנה האפליקציה |
| `style.css` | עיצוב |
| `app.js` | לוגיקה + הגדרות משתמשים |
